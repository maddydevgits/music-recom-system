# Softwares
1. VS Code
2. Python3
3. git

# Packages
1. pip install pandas
2. pip install numpy
3. pip install sklearn

5. pip install seaborn

1. Install Softwares
2. Install Packages
3. Clone Repo
4. run ipynb through VS Code
